REST API

We have our "CRUD" operations in the "RestService". The "RestService" has access to the "Facade" class, which has a connection to the database. The RESTful API is just a simple collection of URIs, HTTP calls to those URIs and some JSON representations of resoursces, many of which contain relational links. Each resouse has its own address or URI - every thing the server can provide is exposed as a resource.

JSON Format

JSON (JavaScript Object Notation) is a data-interchange format. It is easy to read and write. It is easy for machines to parse and generate. It is based on a subset of the Javascript Programming Language. JSON is a collection of name/value pairs.
